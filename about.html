About the Museum

Places of great historical importance and cultural value can be found throughout the globe.However,not all people have access to them.Therefore,there is a need for an easily accessible,user-friendly ,widely accessible,cost - free way of informing those individuals about such structures, such as museums that can potentially be located in another continent.This project was developed with that need in mind,and will produce a navigable 3D visualization of the metropolitan museum of art ,found in New York City,which will be running in all modern internet web browsers such as Apple Safari,Google Chrome,Mozilla Firefox and Opera, without the need for the installation of additional plug-ins and without any requirements on the operating system the user is running,since the only requirement from the user is a compatible web browser.The 3D visualization is designed with the average non-technical person in mind,and is controllable using simple mouse gestures and keyboard combinations.
